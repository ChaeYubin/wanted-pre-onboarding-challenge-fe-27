# 원티드 프리온보딩 챌린지 프론트엔드 코스 11월

## 프로젝트 소개

[원티드 프리온보딩 챌린지 사전 과제](https://github.com/starkoora/wanted-pre-onboarding-challenge-fe-1-api)로 제작한 투두리스트 애플리케이션입니다.

## 구현 기능

- 회원가입 및 로그인 기능
- 투두리스트 추가 및 수정, 삭제 기능

## 개발 환경

- React + Vite + Typescript
- react-router-dom
- Zustand
- Axios
- TailwindCSS + shadcn/ui
- ESLint, Prettier
